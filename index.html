私有项目 Data Intelligence

Text2SQL

自然语言到 SQL 的数据智能项目，重点不是只生成语句，而是把查询理解、执行反馈和结果验证串成可评估链路。

  • Schema linking：把自然语言问题对齐到真实表结构
  • SQL 生成：面向业务查询生成可执行语句并控制方言约束
  • 执行校验：通过数据库反馈检查语法、字段和结果可用性
  • 评测闭环：沉淀评测集、错误归因和结果质量验证

私有仓库

AgentRadar 架构图

信号源 -> 评分 / 排序 -> 证据链 -> 周趋势判断 -> 日报 / 月报 / 知识卡输出

  1. 信号源

    Public Signals

    GitHub、项目发布、产品动态、社区讨论与生态新闻。

  2. 清洗归一

    Normalize

    去重、时间窗口、实体归并、元数据补齐。

  3. 评分 / 排序

    Score & Rank

    结合新鲜度、增长、影响力和相关性生成排序理由。

  4. 证据链

    Evidence Chain

    保留来源链接、判断依据、摘要和复核上下文。

  5. 周趋势判断

    Weekly Judgment

    识别阶段性变化，形成有理由、有置信度的趋势结论。

  6. 输出

    Outputs

    日报、月报、项目卡片、知识卡和趋势复盘。
